# mygmssl

##项目介绍
本人的毕业设计内容：基于嵌入式平台的国密算法与随机数生成  
分为三个部分
* 1.在树莓派平台上国密算法的搭载
* 2.使用STM32f407开发板，编写随机数生成算法，替换国密算法中随机数的生成
* 3.编写web api，调用树莓派上的国密算法
